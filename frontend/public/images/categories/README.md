# Category images

Add your category images here with these exact filenames:

- `necklace.jpg` — Necklace
- `mangal-sutra.jpg` — Mangal Sutra
- `earrings.jpg` — Earrings
- `bracelets.jpg` — Bracelets

Square or near-square images work best. Until you add them, the section shows a maroon placeholder with the category name.
